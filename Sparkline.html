SVG sparkline demo

Deze demo bevat een custom element dat een sparkline rendert in SVG. De lijnkleur komt uit CSS via currentColor, de area fill volgt dezelfde kleur, hover toont de actuele waarde en de lijn bouwt geanimeerd op. Het element vuurt spark-hover en spark-leave events uit, dus het is direct bruikbaar met DataStar.

DataStar integratie

De component gebruikt attributes als input en bubbelt custom events naar boven. Dat past goed bij DataStar.

<div

data-signals="{series:[12,16,11,18,20], labels:['ma','di','wo','do','vr'], hover:null}" data-on:spark-hover="$hover = evt.detail" data-on:spark-leave="$hover = null" > 


 :

Belangrijkste API

  • values: JSON array of komma-gescheiden numerieke lijst
  • labels: optioneel, JSON array of komma-gescheiden lijst
  • min / max: optioneel vaste schaal
  • currentColor: lijn en area nemen automatisch de host-kleur over
  • spark-hover: detail bevat { index, value, label, values }

<svg-sparkline
